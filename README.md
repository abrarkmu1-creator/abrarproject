# Exam Paper Generator

## Overview
The Exam Paper Generator is a tool designed to automate the process of creating exam papers for educational institutions. This project aims to alleviate the workload of educators and improve the efficiency of exam preparation.

## Features
- **User-friendly Interface:** Simple and intuitive user interface for ease of use.
- **Customizable Templates:** Ability to customize question paper templates according to the subject and institution's requirements.
- **Question Bank:** Maintain a repository of questions categorized by subject, level, and topic.
- **Random Question Selection:** Randomly select questions from the question bank to generate unique papers.
- **Format Options:** Output exam papers in various formats (PDF, Word, etc.).
- **Preview Option:** Preview exam papers before finalizing them.
- **Export & Print:** Options to export the generated paper and print directly from the application.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abrarkmu1-creator/abrarproject.git
   ```
2. Navigate to the project directory:
   ```bash
   cd abrarproject
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
- Start the application:
  ```bash
  npm start
  ```
- Follow the on-screen instructions to create an exam paper.

## Customization
- You can modify the default templates and styles located in the `templates/` folder.
- Add your own questions to the question bank by editing the files in the `questions/` directory.

## Technical Details
- **Stack:** The application is built using Node.js for the backend and React.js for the frontend.
- **Database:** Utilizes MongoDB for storing questions and user data.
- **Version Control:** The project is maintained using Git for version control.
